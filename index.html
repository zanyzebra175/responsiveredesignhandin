Responsive Redesign Project


Part 1: Chosen Website Interface

The website I have chosen to redesign is the Samson Realty Website - Samson realty is a realtor company that leases properties on the East Side of Providence. I chose to redesign this website because the current website is extremely cluttered, without a clear directive for the viewer. It has limited styling, and has too much information present on the home screen page.

List of Current Problems:

- Information regarding housing and property listings is not clearly seperated from Realty information such as employee hiring, advertisements, etc.

- Website is not sized properly, there is blank whitespace on the right on desktop, and when resizing, the website elements do not adjust with the webpage, instead just getting cut off statically.

- For the housing listings themselves, text is too small to read in many places, and color of text with the backsplash of the property images results in hard visibility and limited legibility.

- Since the table elements used to display properties are a fixed size throughout the webpage, every table has multiple missing elements, with empty headers within the table and no images where there should be due to this specific layout.

- Due to the limited space available for each property listing, limited to no information can be presented about each individual property. When clicking on some images to get more information, only a certain few redirect to a page with more information, some include broken or nonexistent links.

- Some instances of contact information in the page automatically link to a phone call or email, but other duplicated instances do not link to the appropriate contact links, despite the design around it implying that the information serves as as link.

WebAIM WAVE Findings:

WebAIM WAVE found an abundance of Accessibility problems with this webpage, including but not limited to:

- 34 Images without any alternative text present, making navigation especially difficult for users that may depend on alternative text.

- 5 Empty links, that are indicated as links but do not link to any external or internal pages.

- 2 Missing Page Titles, which can be seen by the lack of identifying information or distinguishment between differnet sections of the website.

- 43 Layout Table Alerts, referring to poor table structures with many missing elements. or improper sizing between different table elements.

I agree with the errors that are identified by WebAIM, and especially agree with the notes on Table Alerts. There are many ways to organize the property information that would allow for the space on the page to be properly used. The links are inconsistent throughout the page, especially with some images linking to external pages and others being static. I also think that the page could strongly benefit from a clear diversification of information, perhaps even seperated out into different pages, as opposed to all information being shown to the user at once.
